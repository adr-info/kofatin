# KÔFATIN - Application de gestion Bar & Dépôt

## Description
Application web complète pour la gestion intégrée d'un bar-dépôt :
- Gestion des ventes (bar et dépôt)
- Gestion des stocks avec conversion casiers/bouteilles
- Gestion des ressources humaines
- Gestion financière (dettes, dépenses, trésorerie)
- Rapports et statistiques

## Technologies utilisées
- PHP 8.0
- MySQL / MariaDB
- Bootstrap 5
- JavaScript
- TCPDF

## Installation
1. Cloner le dépôt
2. Importer la base de données
3. Configurer database/connection.php
4. Accéder à l'application
